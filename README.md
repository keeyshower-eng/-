本项目共有七个文件，spam mail.csv是数据集，其他六个文件是不同的机器学习方法对应的文件，与数据集放在同一个文件夹中即可运行。
每个文件分别需要安装以下库保证运行：
LR-F0.3评估.py：逻辑回归模型，需要：pandas numpy scikit-learn matplotlib seaborn regex
朴素贝叶斯.py：pandas numpy scikit-learn matplotlib seaborn
随机森林.py：pandas numpy scikit-learn matplotlib seaborn
支持向量机SVM.py：pandas numpy scikit-learn matplotlib seaborn 
梯度提升树XGboost.py：pandas numpy matplotlib seaborn scikit-learn xgboost
TextCNN.py：pandas numpy matplotlib seaborn scikit-learn tensorflow
